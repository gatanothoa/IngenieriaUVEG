# Proyecto Udemy - Programación

Este repositorio contiene mis proyectos y ejercicios del curso de programación de Udemy, organizados por lenguaje y día de estudio.

## Estructura del Proyecto

### 📁 JavaScript
- **Día Uno**: Introducción a HTML y JavaScript básico
  - MiPrimeraPagina.html
  - Etiquetas HTML básicas

### 📁 Python
- **Día 1**: Fundamentos de Python
  - Operaciones básicas con print()
  - Manejo de input del usuario
  - Trabajo con strings
  - Primer proyecto

- **Día 2**: Tipos de datos y conversiones
  - Variables y tipos de datos
  - Conversiones entre tipos
  - Formateo de cadenas
  - Redondeo de números

- **Día 3**: Estructuras de datos
  - Listas y métodos
  - Diccionarios
  - Sets y tuples
  - Métodos de strings
  - Extracción de substrings

- **Día 4**: (En desarrollo)

### 📁 SQL
- Primeras bases de datos
- Esquemas y datos de Netflix

### 📁 Recursos
Documentación en PDF sobre todos los temas cubiertos.

## 🚀 Cómo usar este repositorio

1. Clona el repositorio
2. Navega a la carpeta del lenguaje que te interese
3. Revisa los ejercicios día por día
4. Consulta los recursos PDF para más información
